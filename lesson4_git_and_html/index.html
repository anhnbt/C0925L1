Nhớ mùa thu Hà Nội

Hà Nội mùa thu, cây cơm nguội vàng, cây bàng lá đỏ,
nằm kề bên nhau, phố xưa nhà cổ, mái ngói thâm nâu.

Hà Nội mùa thu, mùa thu Hà Nội,
mùa hoa sữa về thơm từng ngọn gió,
mùa cốm xanh về, thơm bàn tay nhỏ,
cốm sữa vỉa hè, thơm bước chân qua.


Hồ Tây chiều thu, mặt nước vàng lay bờ xa mời gọi.
Màu sương thương nhớ, bầy sâm cầm nhỏ vỗ cánh mặt trời.

Hà Nội mùa thu đi giữa mọi người1,
lòng như thầm hỏi, tôi đang nhớ ai2,
sẽ có một ngày trời thu Hà Nội trả lời cho tôi,
sẽ có một ngày từng con đường nhỏ trả lời cho tôi.

Hà Nội mùa thu, mùa thu Hà Nội, nhớ đến một người ...
Để nhớ mọi người.

  • Bước 1: Tạo kho lưu trữ local repo (git init)
  • Bước 2: Đưa file vào vùng theo dõi (git add . hoặc git add tên_file)
  • Bước 3: Tạo snapshot: git commit -m "First commit"
  • Bước 4: Đổi tên nhánh master -> main (git branch -M main)
  • Bước 5: Gắn local repo vào remote repo: git remote add origin https://github.com/anhnbt/demo.git
  • Bước 6: Thực hiện đồng bộ lên remote: git push -u origin main
